📊 CONFIGURATION RAPIDE - Google Analytics 4
=============================================

🚀 POUR PASSER AUX VRAIES DONNÉES EN 5 MINUTES :

1️⃣ CRÉER COMPTE GOOGLE ANALYTICS
   → Aller sur https://analytics.google.com/
   → Créer une propriété "Portfolio Bouba"
   → Noter votre MEASUREMENT_ID (ex: G-ABC123XYZ)

2️⃣ MODIFIER LE FICHIER real-analytics.js
   → Ouvrir real-analytics.js
   → Ligne 189 : Remplacer 'G-XXXXXXXXXX' par votre vrai ID
   → Sauvegarder

3️⃣ DÉPLOYER LE SITE
   → Le système va automatiquement basculer en mode "production"
   → Les vraies données apparaîtront dans 24-48h

4️⃣ VÉRIFICATION
   → Badge devient vert "Données réelles"
   → Console browser affiche "Google Analytics 4 chargé avec succès"

🎯 FONCTIONNALITÉS ACTIVÉES :
✅ Visiteurs en temps réel
✅ Pages vues totales  
✅ Géolocalisation automatique
✅ Événements personnalisés (clics projets, navigation, contact)
✅ Mode hybride (switch démo/production avec Ctrl+Alt+A)

⚡ BONUS - ÉVÉNEMENTS TRACKÉS :
• Clics sur projets
• Navigation entre sections  
• Clics sur liens sociaux/contact
• Temps passé sur chaque section

🎨 DESIGN CONSERVÉ :

• Visiteurs actuels (estimation)

• Visiteurs uniques (session tracking)

📊 DONNÉES COMPLÈTES APRÈS 24-48H :
• Analytics géographiques précises
• Sources de trafic
